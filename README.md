# PhotoApp

PhotoApp is a simple Android application built with Kotlin that showcases a gallery of images in a clean, grid-based layout. The app starts with a splash screen and then presents a list of photos fetched from a remote source.

## Features

*   **Splash Screen:** A welcoming splash screen with the app's name and a progress indicator.
*   **Image Gallery:** Displays photos in a responsive 2-column grid.
*   **Modern UI:** Built with modern Android components like `RecyclerView` and `ViewModel`.
*   **Remote Data:** Fetches image data from a remote repository.
*   **Action Bar:** Includes a top action bar with the app title and an exit button.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Android Studio
*   An Android device or emulator

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Ashish7428/PhotoApp.git
    ```
2.  Open the project in Android Studio.
3.  Build and run the project on your selected device.
